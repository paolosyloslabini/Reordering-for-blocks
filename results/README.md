# results/

Aggregated experiment results in CSV format.

## Files

### results_analysis.csv

Matrix structural metrics computed by `MtxPerm/ANALYSIS/` tools.

**Key Columns:**
| Column | Description |
|--------|-------------|
| `matrix` | Matrix name |
| `perm` | Permutation algorithm applied |
| `perm_type` | ROW, SYMMETRIC, or ASYMMETRIC |
| `bandwidth_max` | Maximum bandwidth |
| `bandwidth_avg` | Average bandwidth |
| `block_density_4` | Block density at block size 4 |
| `block_density_8` | Block density at block size 8 |
| `block_density_16` | Block density at block size 16 |
| `block_density_32` | Block density at block size 32 |
| `block_density_64` | Block density at block size 64 |
| `block_density_128` | Block density at block size 128 |
| `locality_avg_row_spread` | Average nonzero spread per row |
| `locality_vertical_adjacency_ratio` | Consecutive column pairs ratio |

### results_operations.csv

SpMM benchmark timings.

**Key Columns:**
| Column | Description |
|--------|-------------|
| `matrix` | Matrix name |
| `kernel` | SpMM kernel name (e.g., CUSPARSE_SPMM_CSR) |
| `perm` | Permutation algorithm |
| `perm_type` | ROW, SYMMETRIC, or ASYMMETRIC |
| `n_cols` | Dense matrix width (32, 256, 1024) |
| `time_operation_ms` | Execution time in milliseconds |
| `time_reorder_ms` | Time to apply permutation |
| `nnz` | Number of non-zeros |

## Generated By

```bash
python scripts/parse_results.py
```

Parses output from `SbatchMan/experiments/*/jobs/*/output.txt`.

## Usage in Plotting

```python
import pandas as pd

analysis = pd.read_csv('results/results_analysis.csv')
operations = pd.read_csv('results/results_operations.csv')

# Merge for correlation analysis
merged = operations.merge(analysis, on=['matrix', 'perm', 'perm_type'])
```
